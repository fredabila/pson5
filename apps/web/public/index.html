Workflows

Use PSON5 as the cognition layer behind agents, products, and internal tools.

1. Learn

Collect user signals through adaptive prompts and product interactions.

2. Model

Derive stable traits, heuristics, states, and a knowledge graph with evidence and confidence.

3. Project

Give agents only relevant personalization context through a standardized agent view.

4. Simulate

Predict likely user responses, actions, and friction points before the agent acts.
