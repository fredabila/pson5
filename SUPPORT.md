# Support

## Where To Start

- Usage and integration questions: read [README.md](README.md) and [docs/](docs/)
- Bug reports: open an issue at [github.com/fredabila/pson5/issues](https://github.com/fredabila/pson5/issues)
- Security issues: follow [SECURITY.md](SECURITY.md)

## Good Support Requests

Include:

- the package or app involved
- the command or API route you used
- expected behavior
- actual behavior
- relevant logs or stack traces

## Not A Support Channel

This repo is not a place to submit raw personal data, production secrets, or real user profile exports.
