# Support

## Where To Start

- Usage and integration questions: read [README.md](/C:/Users/user/pson5/README.md) and [docs/](/C:/Users/user/pson5/docs)
- Bug reports: open a GitHub issue
- Security issues: follow [SECURITY.md](/C:/Users/user/pson5/SECURITY.md)

## Good Support Requests

Include:

- the package or app involved
- the command or API route you used
- expected behavior
- actual behavior
- relevant logs or stack traces

## Not A Support Channel

This repo is not a place to submit raw personal data, production secrets, or real user profile exports.
