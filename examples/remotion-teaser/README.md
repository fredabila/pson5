# PSON5 · Remotion teaser

A 60-second product teaser for PSON5, built in [Remotion](https://remotion.dev). Matches the landing page's dark-editorial aesthetic exactly — Fraunces display serif, Inter for UI, JetBrains Mono for labels, phosphor-green accent, and the three-colour observed / inferred / simulated grammar running through every scene.

See [`storyboard.md`](storyboard.md) for the shot-by-shot plan.

> **v2 refresh** · The teaser now uses real device mockups (titanium iPhone frame with Dynamic Island, macOS window chrome with traffic lights), iMessage-style bubbles with typewriter reveal, ambient phosphor particles drifting across every scene, and coloured scene-wipe transitions between beats. The three-layer lanes tilt in perspective; the outro orbits the wordmark with four concentric rings and satellite dots.
>
> **v2 also adds** a knowledge-graph scene (force-directed SVG wiring observed → inferred → simulated), a benchmarks scene (three real median latencies pulled from a reproducible `npm run bench`), and a fully procedural soundtrack (ambient bed + whoosh + tick + impact + chime, all generated by `npm run build:audio`). Runtime now ~74 seconds.

## Prerequisites

- **Node.js 20 or newer**
- **Chromium** — installed automatically by Remotion on first render, but on some Linux hosts you may need system deps: `apt install -y libnss3 libatk-bridge2.0-0 libxkbcommon0 libasound2`.

## Install

```bash
cd examples/remotion-teaser
npm install
```

First install pulls ~300MB (Remotion + Chrome headless). It's a standalone project — deliberately not a workspace member of the main monorepo — so the video tooling doesn't pollute the PSON5 engine builds.

## Generate assets (audio + benchmarks)

All audio and benchmark numbers are produced by local scripts — nothing ships as a binary blob.

```bash
npm run build:music   # → public/audio/ambient.wav (Lyria 2 via Vertex AI, stitched 3×30s with crossfade)
npm run build:sfx     # → public/audio/{whoosh,tick,impact,chime}.wav (procedural DSP)
npm run build:audio   # runs both above
npm run bench         # → public/benchmarks.json (median latencies, real measurements)
npm run build:assets  # runs music + sfx + bench
```

`npm run studio`, `npm run render`, and `npm run render-hq` all run `build:assets` automatically, so after the first `npm install` you only need to re-run the generators when you tweak them.

### Music generation (Google Lyria 2)

The ambient bed is generated by Google DeepMind's Lyria 2 model via Vertex AI. One-time setup:

```bash
gcloud auth application-default login --project=<YOUR_PROJECT_ID>
gcloud services enable aiplatform.googleapis.com --project=<YOUR_PROJECT_ID>
```

Then either set `PROJECT_ID=<your project>` in the environment or edit the default in `scripts/generate-music-lyria.mjs`. The script caches Lyria's response at `.cache/lyria/<hash>.json`, so subsequent runs with the same prompt are free and offline. Pass `REFRESH=1` to bypass the cache.

If you don't have a Google Cloud project, skip `build:music` — the teaser still renders, just without the music bed (SFX still play). Or swap in a licensed bed of your own at `public/audio/ambient.wav`.

## Preview in the browser

```bash
npm run studio
```

Opens Remotion Studio at `http://localhost:3000`. You can scrub the timeline, edit any scene, and see changes hot-reload.

## Render the final MP4

```bash
# Default (CRF 23, concurrency 2, ~3-5 minutes on a modern laptop)
npm run render

# High quality (CRF 16, concurrency 4, larger file)
npm run render-hq
```

Output lands at `out/pson5-teaser.mp4`. 1920×1080 H.264, `yuv420p`, 30fps. Good for Twitter, LinkedIn, and web embeds.

## Retime the whole piece

Edit `src/style/tokens.ts` → `SCENE` table. Every `Sequence` in `Teaser.tsx` reads from that single source, so shifting one scene reflows everything else automatically.

```ts
export const SCENE = {
  hook: { from: 0, duration: 180 },
  genericWound: { from: 180, duration: 300 },
  threeLayers: { from: 480, duration: 360 },
  loopInAction: { from: 840, duration: 300 },
  decision: { from: 1140, duration: 300 },
  tagline: { from: 1440, duration: 180 },
  outro: { from: 1620, duration: 180 }
} as const;
```

## Change the copy

Script lines live inline in each `scenes/*.tsx` file as plain strings — swap the English without touching any motion code.

## Change the colours

All tokens are in `src/style/tokens.ts`. Swap `accent` to change the phosphor green. Swap `observed` / `inferred` / `simulated` to change the layer palette (but keep the amber → green → blue progression; it echoes the three epistemic states).
